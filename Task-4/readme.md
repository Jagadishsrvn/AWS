# AWS S3 Bucket with CloudWatch Logging

##  Objective
Create an S3 bucket with no public access, upload files securely, and view access logs using Amazon CloudWatch.

## Steps Performed:
# 1. Create an S3 Bucket
# 2. Upload Files
# 3. Create CloudTrail
# 4. Enable CloudWatch Logging for S3 in CloudTrail
# 5. Verify Logs in CloudWatch
