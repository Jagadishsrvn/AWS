# AWS Project: Windows EC2 Instance

## Objective
To create a Windows Virtual Machine (EC2 Instance) in AWS, connect to it using RDP, open Command Prompt (CMD), and display the system information.

## Steps
1. Launch EC2 Instances
2. Connect using RDP
3. Open CMD in Virtual Machine
4. Type systeminfo in CMD

## Output
- Windows VM connect with RDP
- System information of Virtual Machine
