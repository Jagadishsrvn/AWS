# AWS EC2 and EBS 

## Objective
Launch EC2 instances Linux, configure Apache web servers, attach EBS volumes and snapshots

## Steps
1. Launch EC2 Instances
2. Install Web Servers
3. Create and Attach 5GB EBS Volume
4. Take Snapshot
5. Create New Volume from Snapshot

## Output
- Apache Web Server on Linux
- IIS Web Server on Windows
- EBS Snapshot and Volume Management
